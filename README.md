ShasthoSheba is a mobile health app designed to make healthcare accessible in rural Bangladesh. It helps users track mental health, find nearby clinics, connect with health workers, and get timely health informationâ€”all while working offline and respecting local culture.
Problem We're Solving
Rural Bangladesh faces major healthcare challenges:
Mental health stigma and lack of awareness
Healthcare facilities are far away (80+ km in many areas)
Poor internet connectivity and frequent power cuts
Low digital literacy among older adults
Lack of organized health information
Features
1. Mental Health Check-In
Daily mood tracker with Bengali prompts. Data stays private on your device.
2. Community Health Map
Find nearby clinics, pharmacies, and health workers. Works offline with landmark-based directions.
3. Anonymous Help Request
Get mental health support anonymously through local NGOs. No personal info required.
4. Seasonal Health Tips
Get timely health advice: dengue prevention in monsoon, flu care in winter, etc.
5. Maternal & Child Health Tracker
Track vaccination schedules and ANC visits following Bangladesh EPI calendar.
6. Symptom Guide
Learn when symptoms need medical attention. Educational only, not a diagnosis tool.
7. Health Events Calendar
Stay updated on health camps, vaccination drives, and blood donation events in your area.
8. Health Worker Directory
Find verified community health workers with their skills and contact info.
9. Voice Navigation
For low-literacy users: speak in Bangla, app reads information aloud.
10. NGO Analytics
Privacy-safe aggregated data export for NGOs to track community health trends.
Tech Stack
React + Tailwind CSS
Web Speech API for voice features
Geolocation API for maps
Local storage for offline functionality
